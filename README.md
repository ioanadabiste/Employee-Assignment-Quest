# Employee-Assignment-Quest
Task Management System – Java

A simple yet structured Task Management System built in Java, designed using object-oriented programming principles.
##  Features  

-  **Core OOP Concepts:** Inheritance, Polymorphism, and Abstraction  
-  **Multiple Task Types:** `SimpleTask` and `ComplexTask`  
-  **Employee–Task Assignment Logic**  
-  **Reusable Utility Classes:** Data validation, serialization, and file handling  
-  **Persistent Data Storage** using serialization  
-  **Safe Data Management:** Collections and exception handling  
-  **Modular Architecture** for clean, maintainable, and scalable code
